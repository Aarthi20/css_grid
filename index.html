CSS Grid

Introduction to CSS

Understand what CSS is, how it works, include CSS in HTML is different ways such that iniline, internal, external, and Basic sytax and selectors

Box Model and Layout

Learn about the box model and how to use it to control layout, Box model(content, padding, border, margin) Display properties(block, inline, inline-block, none).Positioning(static, relative, absolute, fixed, sticky), and Float & Clear

Styling Text and Fonts

Learn to style text effectively, Font properties(font-family, font-size, font-weight, gont-style), Text properties(text-decoration, text-align, text-transform, color, line-height, letter-spacing), and Google Fonts

Advanced Selectors and Combinators

Learn advanced CSS selectors and how to combine them for precise styling, Attribute selectors, Pseudo-classes and Pseudo-elemets, and Combinators(descendant, child, adjacent sibling, general sibling)

Responsive Design and Media Queries

Learn to create responsive web designs, Responsive design principles, Media queries, and Fluid grids and flexible images

CSS Flexbox

Understand and implement CSS Flexbox for layout. Flex container properties (display: flex, flex-direction, flex-wrap, justify-content, align-items, align-content), and Flex item properties (order, flex-grow, flex-shrink, flex-basis, align-self)

CSS Grid and Review

Grid container properties (display: grid, grid-template-columns, grid-template-rows, grid-gap) and Grid item properties (grid-column, grid-row, grid-area)
